
#### List all available versions
```ruby
rbenv install -l
```
#### Install Ruby x.x.x
```ruby
rbenv install x.x.x
```
#### Uninstall Ruby x.x.x
```ruby
rbenv uninstall x.x.x
```
#### See installed versions
```ruby
rbenv versions
```
#### See current version
```ruby
rbenv version
```
#### Display path to executable
```ruby
rbenv which <NAME>
```
#### Re-write binstubs
```ruby
rbenv rehash
````
